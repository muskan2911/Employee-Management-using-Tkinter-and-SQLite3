# Employee-Management-using-Tkinter-and-SQLite3

**Employee Management System**

This program is a simple Employee Management System developed using Python's Tkinter library and SQLite database. It allows users to add, delete, and print employee details.

**Features:**
- **Add Employee:** Enter the employee's name, age, and salary, then click the "Add Employee" button to add the employee to the database.
- **Delete Employee:** Select an employee from the list and click the "Delete Employee" button to remove the employee from the database.
- **Print Employee:** Select an employee from the list and click the "Print Employee" button to display the employee's details.
- **List of Employees:** Displays a list of all employees currently stored in the database.

**How to Run:**
1. Ensure you have Python installed on your system.
2. Execute the provided Python script (`employee_management_system.py`) using any Python interpreter.
3. The main window of the Employee Management System will appear, allowing you to perform various operations on employee records.

**Dependencies:**
- Python 3.x
- Tkinter library (usually comes pre-installed with Python)
- SQLite3 library (usually comes pre-installed with Python)
